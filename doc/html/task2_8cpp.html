Get4Pixels16Bit reads four consecutive pixels of the specified row started from given column and writes they to the two registers out_BG and out_RA. Uses 16 bit per channel

Parameters
in_imgis a input image
in_row_idxis an index of a row to read pixels
in_col_idxia an index of a column with a first pixel
out_BGis a pointer to a 128bit register to store blue and green channels for the pixels four consecutive pixels in format BBBB GGGG. Order of pixels is [0, 1, 2, 3]
out_RAis a pointer to a 128bit register to store red and alpha channels for the pixels four consecutive pixels in format RRRR AAAA. Order of pixels is [0, 1, 2, 3]
